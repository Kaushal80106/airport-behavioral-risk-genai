# GenAI-Assisted Airport Behavioral Risk Analysis System

## Overview
An AI-powered decision support system that analyzes passenger behavior, facial stress indicators, body language, and sentiment to generate risk scores for secondary airport screening.

## Tech Stack
- FastAPI
- Computer Vision (OpenCV, MediaPipe)
- NLP & Sentiment Analysis
- Generative AI (Explainable Risk Reports)
- React (Dashboard)
